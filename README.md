# \# WC Data Automation Project



## Proyecto de automatización de procesamiento de datos operativos utilizando Python.





### Este proyecto transforma procesos manuales en Excel en un flujo automatizado usando Python.

### 

### Permite:

### \- Limpiar y estructurar datos

### \- Aplicar reglas de cálculo por WC Code

### \- Generar reportes automáticos

### \- Reducir errores manuales



###  Tecnologías

### 

### \- Python

### \- Pandas

### \- OpenPyXL

### \- Excel

### 

###  Objetivo

### 

### Demostrar cómo automatizar análisis de datos operativos para mejorar eficiencia y calidad de información.



### *Datos utilizados fueron anonimizados para fines demostrativos.*
